# 📦 スマート在庫管理システム (Zaiko Management System)

AWS上に構築した、Webベースの在庫管理アプリケーションです。
在庫の入出庫ログ、ロット番号（Lot No.）管理、そしてAWS環境（EC2 + Nginx）での安定稼働を想定した構成を実装しました。

## 📷 アプリの画面

## 🛠 使用技術 (Tech Stack)
**バックエンド**
* **Python 3.13.3** (Flask framework)
* **MySQL** (在庫データの永続化)

**インフラ・クラウド (AWS)**
* **EC2** (Amazon Linux 2023 / Webサーバー)
* **Nginx** (リバースプロキシ / Webサーバー設定)


**フロントエンド・その他**
* **HTML / JavaScript** (UI/UXの実装)
* **Git / GitHub** (バージョン管理)
* **python-dotenv** (環境変数によるセキュリティ対策)

#### 🔥 こだわったポイント

* **正確な時間管理 (JST対応)**
    * 在庫管理において「いつ」操作されたかは重要です。システムのデフォルトタイムゾーンを日本標準時 (JST) に設定し、正確なログ記録を実現しました。

* **ロット管理機能 (Lot No.)**
    * 単なる個数管理だけでなく、製品のロット番号 (Lot No.) も追跡可能な機能を実装。より詳細なトレーサビリティを確保しています。

* **コードの構造化 (Refactoring)**
    * `app.py` に混在しがちなHTML/JSロジックを分離し、メンテナンス性と可読性の高いコード設計へリファクタリングを行いました。

* **双方向検索機能 & ログ**
    * 必要な在庫データを即座に見つけられる検索機能と、入出庫の履歴を追える「InventoryLog」機能を備えています。

* **堅牢なインフラ構成**
    * Pythonアプリケーションを直接公開するのではなく、Nginxを前段に置くことで、セキュリティとパフォーマンスを考慮した構成としています。
